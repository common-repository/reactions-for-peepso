== Description ==

Extend the regular Likes with Reactions expressing much more: Love, Laughter, Wink, Wow, Sad, Angry

This plugin is compatible with PeepSo 1.5.7 or newer

== Installation ==
1. This plugin only activates if PeepSo  plugin is present and active
1. Minimum required PeepSo version is 1.5.7
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the PeepSo->Config->Reactions tab to enable the integration

== Screenshots ==
